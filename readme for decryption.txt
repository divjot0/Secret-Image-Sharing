Steps for decryption process:-

1) Run the file "Decryption.cpp". It asks for shadow images as input. It creates an image named "extractedImage.tiff". This image is the permuted version of the decrypted image.

2) Now run the file "Reverse_Permutation.cpp". This file takes input the above obtained file. This will create the final decrypted image as a .tiff image.
